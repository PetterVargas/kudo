---
title: Paso 3 - Ejecución local
description: Levanta el proyecto en tu equipo para verificar que funciona correctamente
---

```bash tab="pnpm"
cd tu-proyecto
pnpm install
pnpm dev
```

```bash tab="npm"
cd tu-proyecto
npm install
npm run dev
```

```bash tab="yarn"
cd tu-proyecto
yarn install
yarn dev
```

Tu aplicación estará disponible en [http://localhost:3000](http://localhost:3000)
