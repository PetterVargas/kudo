---
title: Paso 7 - Subir el código al repositorio
description: Inicializa Git y sube el proyecto a tu repositorio remoto
---

Inicializa Git y sube el proyecto a tu repositorio remoto:

```bash
git init
git add .
git commit -m "feat: initial SGSI setup"
git branch -M main
git remote add origin https://github.com/tu-usuario/tu-repositorio.git
git push -u origin main
```
