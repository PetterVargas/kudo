# CNB - Ciberseguridad en Nube

## Descripción

El dominio de Ciberseguridad en Nube establece controles y mejores prácticas para la seguridad en entornos de nube pública, privada e híbrida.

## Controles

Este dominio incluye controles relacionados con:

- Configuración segura de servicios cloud
- Modelo de responsabilidad compartida
- Gestión de acceso a recursos cloud
- Cifrado de datos en tránsito y en reposo
- Monitoreo y logging en la nube
