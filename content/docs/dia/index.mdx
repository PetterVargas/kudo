# DIA - Ciberseguridad en Datos e Inteligencia Artificial

## Descripción

El dominio de Ciberseguridad en Datos e Inteligencia Artificial establece controles para la protección de datos y el uso responsable de IA.

## Controles

Este dominio incluye controles relacionados con:

- Protección de datos personales (GDPR, LGPD)
- Clasificación y etiquetado de datos
- Cifrado y anonimización de datos
- Gobernanza de inteligencia artificial
- Ética en IA y prevención de sesgos
