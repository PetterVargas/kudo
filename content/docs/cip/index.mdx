# CIP - Ciberseguridad en Identidad y Puntos Finales

## Descripción

El dominio de Ciberseguridad en Identidad y Puntos Finales aborda la gestión de identidades, autenticación, autorización y seguridad de dispositivos.

## Controles

Este dominio incluye controles relacionados con:

- Gestión de identidades y accesos (IAM)
- Autenticación multifactor (MFA)
- Gestión de dispositivos (MDM/EMM)
- Seguridad de endpoints
- Control de acceso basado en roles (RBAC)
