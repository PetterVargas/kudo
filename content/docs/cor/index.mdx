# COR - Coherencia Organizacional

## Descripción

El dominio de Coherencia Organizacional establece las bases de gobernanza y estructura para la gestión de ciberseguridad en la organización.

## Controles

Este dominio incluye controles relacionados con:

- Gobernanza de ciberseguridad
- Estructura organizacional
- Roles y responsabilidades
- Políticas y procedimientos
- Gestión de riesgos
