# ADR - Análisis, Detección y Respuesta de Ciberseguridad

## Descripción

El dominio de Análisis, Detección y Respuesta establece capacidades para identificar, analizar y responder a incidentes de ciberseguridad.

## Controles

Este dominio incluye controles relacionados con:

- Security Operations Center (SOC)
- Security Information and Event Management (SIEM)
- Análisis de amenazas (Threat Intelligence)
- Respuesta a incidentes (Incident Response)
- Análisis forense digital
