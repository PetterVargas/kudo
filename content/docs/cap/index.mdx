# CAP - Ciberseguridad en Aplicaciones

## Descripción

El dominio de Ciberseguridad en Aplicaciones se enfoca en el desarrollo seguro de software y la protección de aplicaciones contra vulnerabilidades.

## Controles

Este dominio incluye controles relacionados con:

- Desarrollo seguro (SDLC/DevSecOps)
- Análisis estático y dinámico de código (SAST/DAST)
- Gestión de dependencias y vulnerabilidades
- Pruebas de seguridad de aplicaciones
- Protección contra OWASP Top 10
