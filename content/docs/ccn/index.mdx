# CCN - Continuidad y Cambios del Negocio

## Descripción

El dominio de Continuidad y Cambios del Negocio aborda la gestión de cambios, continuidad operacional y recuperación ante desastres.

## Controles

Este dominio incluye controles relacionados con:

- Gestión de cambios (Change Management)
- Plan de continuidad del negocio (BCP)
- Plan de recuperación ante desastres (DRP)
- Gestión de incidentes
- Análisis de impacto al negocio (BIA)
